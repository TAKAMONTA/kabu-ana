# iPadテストとスクリーンショット生成ガイド

## 概要
iPad実機がない場合でも、Xcodeのシミュレーターを使用してiPadでの動作確認とスクリーンショット生成が可能です。

## 必要な環境
- macOS（Xcodeが動作する環境）
- Xcode（最新版推奨）
- Apple Developerアカウント

## 手順

### 1. Xcodeでプロジェクトを開く

```bash
cd ios/App
open App.xcworkspace
```

### 2. iPadシミュレーターを起動

1. Xcodeのメニューから **Product > Destination** を選択
2. または、Xcodeの上部バーでデバイス選択ドロップダウンをクリック
3. **iPad Air 11インチ (M3)** または **iPad Pro 12.9インチ** を選択
4. シミュレーターが起動します

### 3. アプリをビルドして実行

1. Xcodeで **Product > Run** (⌘R) を選択
2. または、再生ボタンをクリック
3. アプリがシミュレーターにインストールされ、自動的に起動します

### 4. 動作確認

以下の機能をiPadシミュレーターでテストしてください：

#### ✅ 必須テスト項目

1. **今日の注目銘柄セクションの分析ボタン**
   - 各銘柄の「分析」ボタンをタップ
   - エラーが表示されないことを確認
   - 正常に検索結果が表示されることを確認

2. **新規アカウント作成**
   - ログインボタンをタップ
   - 「新規登録」タブを選択
   - メールアドレスとパスワードを入力
   - エラーメッセージが日本語で表示されることを確認
   - 正常にアカウントが作成されることを確認

3. **その他の機能**
   - 検索機能
   - AI分析機能
   - チャート表示
   - 横画面・縦画面の両方で表示確認

### 5. スクリーンショットの撮影

#### 方法1: シミュレーターのスクリーンショット機能を使用

1. シミュレーターでアプリを起動
2. スクリーンショットを撮りたい画面に移動
3. **File > New Screen Recording** または **⌘S** でスクリーンショットを保存
4. 保存された画像は `~/Desktop` に保存されます

#### 方法2: コマンドラインからスクリーンショットを取得

```bash
# シミュレーターのデバイスIDを確認
xcrun simctl list devices

# スクリーンショットを撮影（デバイスIDを指定）
xcrun simctl io booted screenshot ~/Desktop/ipad-screenshot.png
```

#### 方法3: Xcodeのデバッグメニューから

1. シミュレーターでアプリを実行中
2. **Debug > View Debugging > Take Screenshot of [デバイス名]**
3. スクリーンショットが保存されます

### 6. 必要なスクリーンショットサイズ

App Store Connectにアップロードする必要があるサイズ：

#### iPad Pro 12.9インチ（第3世代以降）
- 2048 x 2732 pixels（縦向き）
- 2732 x 2048 pixels（横向き）

#### iPad Pro 11インチ
- 1668 x 2388 pixels（縦向き）
- 2388 x 1668 pixels（横向き）

#### iPad Air 11インチ（M3）
- 1668 x 2388 pixels（縦向き）
- 2388 x 1668 pixels（横向き）

### 7. スクリーンショットの要件

- ✅ 実際のiPadアプリの画面であること（iPhoneの画像を引き伸ばしたものではない）
- ✅ アプリの主要機能を示していること
- ✅ スプラッシュ画面やログイン画面のみではないこと
- ✅ すべての言語で同一の内容であること

### 8. 推奨スクリーンショット内容

1. **メイン画面** - 検索機能と今日の注目銘柄
2. **検索結果画面** - チャートと企業情報
3. **AI分析結果画面** - 分析結果の表示
4. **財務評価画面** - 財務健全性の評価
5. **ニュース分析画面** - ニュース分析結果

### 9. スクリーンショットの編集（必要に応じて）

- 画像編集ソフト（Photoshop、GIMP、Preview等）でサイズ調整
- 必要に応じてフレームを追加（App Store用のデバイスフレーム）

### 10. App Store Connectへのアップロード

1. App Store Connectにログイン
2. アプリを選択
3. **App Store** タブを開く
4. **スクリーンショット** セクションで **iPad Pro 12.9インチ** または **iPad Air 11インチ** を選択
5. スクリーンショットをアップロード

## トラブルシューティング

### シミュレーターが起動しない場合

```bash
# シミュレーターをリセット
xcrun simctl shutdown all
xcrun simctl erase all
```

### アプリがビルドできない場合

```bash
# Podを再インストール
cd ios/App
pod deintegrate
pod install
```

### スクリーンショットのサイズが合わない場合

- 画像編集ソフトでリサイズ
- アスペクト比を維持しながらリサイズ

## 注意事項

⚠️ **重要**: Appleのレビューでは、実際のiPadデバイスでテストされる可能性があります。シミュレーターでのテストに加えて、可能であれば実機でのテストも推奨されます。

⚠️ **スクリーンショット**: iPhoneの画像を引き伸ばしてiPad用として使用することは禁止されています。必ずiPadシミュレーターまたは実機で撮影したスクリーンショットを使用してください。
