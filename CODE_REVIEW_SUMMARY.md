## 🔧 技術的な詳細

### エラーハンドリングの流れ

1. **フロントエンド** (`TopTradingValueSection.tsx`)
   - シンボル形式を検証・変換
   - エラー時はコンソールにログ出力

2. **API層** (`/api/search`)
   - Zodスキーマで検証
   - エラーメッセージを日本語化

3. **フック層** (`useCompanySearch.ts`)
   - APIエラーをキャッチ
   - 日本語メッセージを優先的に取得

4. **UI層** (`page.tsx`)
   - エラーメッセージを表示

### 認証エラーの流れ

1. **Firebase認証** (`useAuth.ts`)
   - Firebaseエラーコードを日本語メッセージに変換

2. **UI層** (`AuthModal.tsx`)
   - エラーメッセージを表示
